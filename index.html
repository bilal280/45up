نبذة عن 45Up

45Up هو نظام إدارة تعلم (LMS) يخدم المدارس والمعلمين بتجربة تعليمية رقمية متكاملة وسهلة الاستخدام لإدارة الصفوف والمتابعة والتحليل.

  • إدارة الصفوف والواجبات والاختبارات من مكان واحد.
  • لوحات متابعة فورية لأداء الطلاب وتحليلات أعمق.
  • تكاملات جاهزة وواجهات برمجية لربط الأنظمة.

المسؤوليات

  • قيادة تطوير تطبيقات الويب باستخدام ASP.NET Core مع الالتزام بـ Clean Architecture.
  • تطوير واجهات تفاعلية باستخدام JavaScript, jQuery, Vue.js.
  • تصميم وإدارة قواعد بيانات معقدة باستخدام SQL Server مع التركيز على الأداء والتوسع.
  • تصميم وتطوير واجهات برمجية REST APIs فعّالة للتكامل مع الأنظمة الخارجية.
  • مراجعة الأكواد وتوجيه المطوّرين الأقل خبرة وتقديم الدعم التقني.
  • المشاركة في تحليل المتطلبات وتصميم الحلول وبنية النظام.
  • ضمان الجودة، الأمان، وتحسين الأداء بشكل مستمر.

  • إلمام بمنهجيات Agile/Scrum.
  • حل المشكلات والتفكير التحليلي.
  • قدرة على القيادة والتعاون مع فرق متعددة التخصصات.
